# Script Editor Quick Guide

## Goal

Write theatre scripts in a centered, touch-friendly editor with minimal controls.

## Screen layout

- Top area: rendered script (largest section).
- Bottom area: compact command panel.
- Mobile keyboard appears only when you tap a script block.

## Core workflow

1. Tap an existing block to edit it.
2. Tap the blank area at the bottom to insert a new paragraph.
3. Use character buttons to create a new dialogue paragraph.
4. Use **Insert action paragraph** to add a stage-direction block.
5. Switch between **Normal** and **Description** at any time.

## Formatting behavior

- Dialogue is always centered and starts with uppercase character cue.
- Description mode wraps text as parenthetical and renders italic style.
- Action paragraphs are styled as descriptive stage directions.

## Metadata editing

- Tap the metadata card at the top of the editor.
- The app opens the wizard in metadata-edit mode.
- Save changes and return with **Back to editor**.

## Mobile notes

- Keyboard opens when a text block gets focus.
- Use **Hide keyboard** (or native back gesture) to dismiss it.
- Keep the panel small and rely on direct tapping for speed.
